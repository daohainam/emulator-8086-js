The Machine That Started It All

Meet the Intel 8086 — the chip that runs inside this browser tab right now

You're running a 1978 computer in your browser

In 1978, Intel released a microprocessor called the Intel 8086. It powered the very first IBM PCs. Every modern x86 chip — your laptop, your server, your gaming PC — is a direct descendant of it. This emulator lets you run 8086 code right here, right now, in JavaScript.

📅

Born in 1978

Intel 8086 introduced the x86 architecture — still alive in every Windows PC today, 45+ years later.

💾

1 Megabyte of RAM

The 8086 could address exactly 1,048,576 bytes (1 MB). Modern chips address billions of times more.

🌐

100% in JavaScript

This emulator faithfully re-creates the 8086 hardware in JS — every register, every flag, every memory byte.

Why does this matter for you?

Every modern computer — your laptop, your phone — still follows this same pipeline: text → compiler/assembler → machine code → CPU → output. The 8086 is the simplest possible example of this universal pattern. Understand the 8086, understand computing.

Three files power this entire emulator

The whole system is built from just three source files:

CPU8086.js

The CPU Brain

~2,500 lines. Implements the hardware: registers, flags, 1 MB memory, and a binary decoder that executes every 8086 instruction.

Assembler8086.js

The Translator

~500 lines. A two-pass NASM-compatible assembler that converts human-readable assembly instructions to binary machine code bytes.

App.jsx

The Control Room

~2,000 lines. The React UI — code editor, VGA monitor, register display, memory viewer, disk viewer, and the run/step loop.
